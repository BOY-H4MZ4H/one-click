# INSTALL
```
$ pkg install python git -y
$ pip install bs4
$ pip install requests
$ pip install rich
$ pip install cython
$ git clone https://github.com/BOY-H4MZ4H/one-click
$ cd $HOME/one-click
$ ls
$ python run.py
```
